==== EXPLICATION ====
Explication du contenu du zip :
- Dans l'archive, se trouve deux fichiers .jar executables , permettant de lancer l'application , et le code source.
1) Le jar Serveur attend un calcul afin de renvoyer le resultat.
2)le jar Client envoie le calcul et attend un resultat .
Le client saisit ses operations de la forme " OPERATION CHIFFRE1 CHIFFRE2" , et le serveur lui renvoie la reponse.
3) le code source se situe dans le dossier "src"
=====================



==== INSTRUCTIONS ====
Pour lancer le programme, il faut :

1) Dezipper l'archive
2 Ouvrir un terminal et taper la commande suivante :
	java -jar serveur.jar
Celle ci permettra d'executer le serveur
3) Ouvrir un terminale et taper la commande suivante :
	java -jar client.jar
Celle ci permettra de se connecter au serveur
4) Entrer des operations dans la console eclipse, de type " + 8 4 " , " * 3 4 " ou encore "+ 2" ...
5) Le resultat s'affiche dans la console
6) pour quitter depuis l'interface client, tapez "quit"

Ces commandes devront être lancé dans 2 terminals differents dans l'ordre indiqué.
